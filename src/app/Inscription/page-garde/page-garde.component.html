A propos de nous

CPM est une organisation qui s'occupe du management des projets, elle peut vous aidez à améliorez vos projets grace à les statistiques fournit pour chaque projet ainsi chaque marchée

  • gestion des marchées
  • gestion des bon de commandes
  • gestion des articles

L'inscription dans CPM est par trois etapes :

  • envoyez une demande
  • si votre demande est validé,votre admin recevra un email qui contien un code de confirmation
  • grace a ce code de confirmation et l 'email de l'admin vous pouvez créer un compte MYCPM et votre organisation sera officiellement inscrite dans CPM
